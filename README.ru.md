## Игровой процесс

- Выберите размер поля — 6x6, 8x8 (классический) или 10x10 — в любой момент через верхнюю панель.
- Внизу экрана всегда доступны три случайные фигуры в лотке.
- Взять фигуру можно любым удобным способом (оба работают взаимозаменяемо, см. [Управление](#управление)):
  - **Прилипание к курсору**: кликните по фигуре (или нажмите `1`/`2`/`3`) — она "прилипнет" к курсору, затем кликните по полю для подтверждения.
  - **Классическое перетаскивание**: зажмите фигуру и сразу тащите её на поле, затем отпустите.
- Пока фигура следует за курсором, клетки, куда она встанет, подсвечиваются зелёным (можно разместить) или красным (клетки заняты или фигура не помещается), а при приближении к валидной позиции она мягко "подтягивается" к ближайшей корректной клетке.
- Отпустите/подтвердите фигуру, чтобы разместить её. Как только все три фигуры из лотка использованы, появляются три новые случайные фигуры — игра гарантирует, что хотя бы одну из них можно будет куда-то поставить, если на поле вообще есть подходящее место.
- Полностью заполненная строка или столбец очищаются с анимацией вспышки и затухания.
- Если несколько очисток происходят подряд (без хода, который ничего не очистил), это засчитывается как комбо — на экране появляется надпись "Combo x2", "Combo x3" и т.д., увеличивающая бонусные очки.
- Игра заканчивается, когда ни одну из трёх текущих фигур невозможно разместить на поле. Появляется экран Game Over с итоговым счётом и кнопкой для новой игры.

## Фигуры

Используется точный классический набор Block Blast: одиночные клетки, линии из 1–5 клеток (по горизонтали и вертикали), квадраты 2x2 и 3x3, сплошной прямоугольник 2x3/3x2, L/J-образные фигуры, T-образные, зигзаги (S/Z), маленькие уголки из 3 клеток (выпадают чаще, как и в оригинале) и большие уголки из 5 клеток. Каждая фигура окрашена в случайный яркий цвет.

## Система очков

- +1 очко за каждую клетку, занятую размещённой фигурой.
- +10 очков за каждую очищенную строку/столбец, умноженные на текущий множитель комбо.
- Множитель комбо увеличивается при каждом ходе, который очищает хотя бы одну линию, и сбрасывается, если ход ничего не очистил.

## Управление

| Действие | Ввод |
|---|---|
| Взять фигуру (прилипание к курсору) | Кликните по фигуре в лотке или нажмите `1` / `2` / `3` |
| Подтвердить размещение (режим прилипания) | Кликните по полю в нужном месте |
| Взять и перетащить фигуру (классический способ) | Зажмите фигуру, тащите на поле, отпустите |
| Сменить размер поля | Нажмите кнопку размера (например **8x8**, верхняя панель) и выберите 6x6 / 8x8 / 10x10 — начнётся новая игра |
| Сменить язык | Нажмите кнопку **RU / EN** (правый верхний угол) |
| Начать заново после Game Over | Нажмите **Начать заново** |

## Локализация

🇬🇧 English · 🇷🇺 Русский

Интерфейс запускается на языке, соответствующем системной локали Windows (русский — если язык интерфейса Windows русский, иначе английский), и может быть переключён в любой момент кнопкой языка в правом верхнем углу.

## Лучший результат

Лучший результат сохраняется автоматически в `%AppData%\BlockBlast\save.json` и сохраняется между запусками.

## Сборка и запуск из исходников

Требуется: [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0), Windows.

```powershell
dotnet build BlockBlast.sln -c Debug
dotnet run --project BlockBlast/BlockBlast.csproj
```

## Публикация самодостаточного exe

Создаёт единый файл `BlockBlast.exe`, который запускается на машине без установленного .NET:

```powershell
dotnet publish BlockBlast/BlockBlast.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o BlockBlast/bin/publish
```

## Сборка установщика для Windows

Требуется [Inno Setup 6](https://jrsoftware.org/isinfo.php).

1. Опубликуйте самодостаточный exe (см. выше), чтобы появился файл `BlockBlast/bin/publish/BlockBlast.exe`.
2. Соберите инсталлятор:

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer\setup.iss
```

Инсталлятор появится в `dist\BlockBlast-Setup.exe`. Он устанавливает игру в Program Files, добавляет ярлык в меню "Пуск", по желанию — ярлык на рабочем столе, и регистрирует деинсталлятор в разделе "Приложения и возможности".
